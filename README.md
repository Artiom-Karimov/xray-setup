# Разворачиваем VLESS-соединение через два VPS

Мы используем две виртуальные машины для организации защищённого удалённого доступа.
Первая машина выполняет роль промежуточного узла, а вторая находится в сети, к которой требуется подключение.
Соединение между узлами организовано через WebSocket и TLS.

## Создаём виртуальные машины

Нам нужно настроить 2 виртуальные машины.

Для этого подойдёт [vdsina](https://vdsina.ru/) и самый дешёвый тариф:

Стандартные серверы -> debian 13 trixie -> 1xCPU, 1 GiB RAM, 10 GiB ROM.

Создаём машины, получаем их IP для следующего шага.

## Получаем доменное имя

Для подключения нам понадобится 2 имени домена - столько же, сколько будет виртуальных машин.

Бесплатно можно получить на [freedns](https://freedns.afraid.org/):

Выбираем subdomain, придумываем адрес, вписываем IP своей VPS.

## Настраиваем клиент
### Скачиваем
Самый популярный клиент - [V2RayN](https://github.com/2dust/v2rayN/releases).

### Настраиваем подключение

- Configuration -> Add VLESS
- Address: `доменное.имя.прокси`
- Port: `443`
- UUID: Содержимое поля `"id"` из `xray/config.json` основной машины
- Transport: `WS`
- Path: `/vlessws`
- TLS: `tls`
- SNI: `доменное.имя.прокси`
- Allow Insecure: `false`
